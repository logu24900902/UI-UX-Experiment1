# UI-UX-Experiment1

## Aim:
To study and understand the fundamentals of User Interface (UI) and User Experience (UX), differentiate between them using real-world applications

## Algorithm:

### 1.Study the basic concepts of User Interface (UI) and User Experience (UX).

### 2.Select real-world applications such as Swiggy and WhatsApp for analysis.

### 3.Identify and observe the UI elements (buttons, layout, colors, icons) used in the selected applications.

### 4.Analyze the UX aspects such as ease of use, navigation, response time, and user satisfaction.

### 5.Design (UI)

### 6.Explore UI/UX design tools such as:
Figma
Adobe XD

### 7.Create a simple prototype or interface using any one tool.

### 8.Test the prototype for usability and user experience.
